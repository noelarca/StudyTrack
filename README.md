# StudyTrack
A simple study tracker, with (in future) some stat functionality

# Requirements
- Python
- PySide6

# Installation
Make sure you have python installed with:
'''
python --version
'''
Else install it here: https://www.python.org/downloads/
Then install PySide6 with:
'''
pip install PySide6
'''
